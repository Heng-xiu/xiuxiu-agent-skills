# 平台演算法速查 + 發文建議（S2 參考 / S5 主用）

> 按需載入：S2（決定挖哪些平台、各平台吃什麼）、S5（給發文建議）。五平台一張表，手機優先。
> FB 細節以 `social-post/references/evaluation.md` 為準（本檔是跨平台速查 + 小紅書補充）。

## 五平台核心訊號對照

| 平台 | 最強訊號 | 次強 | 幾乎無權重 | 一句話策略 |
|---|---|---|---|---|
| **FB** | 陌生人**分享**（20× 讚）| 留言（5×）、dwell time | 讚 | 4段4句長敘事 + 留言 CTA，正文不放連結 |
| **Threads** | **轉發**（identity signal，5-10× 讚）| 高回覆率、keyword | 讚 | 短站隊立場宣言，熱門 keyword 觸發推送 |
| **IG** | **儲存** + Reels 完成率 | 分享到限動 | 讚 | 圖/Reels 必備，首圖即 hook，做可儲存的清單 |
| **X** | **Reply（27× Like）** | 收藏、完讀、首小時互動 | 讚 | Hook 推 + 1/3/1 thread，引發回覆 |
| **小紅書** | 封面點擊率 + **收藏** | 評論、完讀、漲粉 | 讚 | 標題黨封面決定 70% 點擊，乾貨可收藏 |

## 各平台發文建議（S5 逐平台輸出）

### FB
- **時段**：AI/科技受眾夜貓 22:00–01:00（見 social-post R4）；一般受眾午休 12-13、晚 20-22
- **首小時動作**：發完自己在留言區補 1 則「作者精選留言」放 CTA（FB 正文不可連結，留言可放 1 個 URL）；引導前幾位留言者寫長留言（5 字以上 = 3× 權重）
- **格式**：純文字 / 截圖（免費 dwell time）/ 同日配 Reels（+50% 觸及）
- **配額**：一天一篇；爆款後 24h 冷卻禁長文；同敘事意圖 4 天內不重複
- **跨社團**：主帳號發完後 cross-post 到相關社團，留言可放大 5-10×（R30）

### Threads
- **時段**：不太挑時段（演算法 burnout 比 FB 慢），可較頻繁 1-2/天
- **首小時動作**：埋 1-2 個熱門 keyword（你的領域詞）觸發演算法主動推給非追蹤者；回覆早期留言衝回覆率
- **格式**：1 段不換行、連續逗號流、「！」最多到「！！」、60-150 字
- **配額**：立場宣言型每月 2-4 篇 OK

### IG
- **時段**：受眾上線時段（午休 / 晚間 / 睡前 21-23）
- **首小時動作**：限動導流到貼文；引導儲存（「存起來慢慢看」）
- **格式**：carousel（首圖 hook + 後續乾貨）或 15-30 秒 Reels（完成率最佳，比 photo +135% 觸及）；**純文字改發 Threads/X**
- **配額**：2-3/週（必圖）

### X
- **時段**：受眾活躍時段；thread 不挑時段，短推看受眾時區
- **首小時動作**：自己在第一條 reply 補連結（外連移到回覆，避免降權）；引導 reply（27× like）
- **格式**：Hook 推 + 8-12 條正文（每條獨立成文）+ TL;DR + CTA
- **配額**：5-7/週；thread 對打/橫評最易擴散

### 小紅書 ⚠️
- **時段**：中國時區晚 19-23、週末
- **首小時動作**：（手動發）標題 A/B 試、引導收藏（「碼住」）+ 評論區提問
- **格式**：封面標題黨（數字 + 痛點 + 結果）+ 圖文 9 張內 + emoji 分點 + 結尾 5-10 個 # 標籤（蹭分類流量）
- **配額**：可較頻繁，標題決定 70% 成敗
- **發佈**：**無自動發佈 API**，只給草稿 + 手動步驟，不承諾代發

## 📱 手機優先檢查（所有平台共通）

- 首屏只有前 1-2 行 → hook 必須在第一行
- 段落 ≤ 3 行手機螢幕寬
- FB/IG 空行分段；Threads 逗號流不換行
- 圖/影片首幀即封面即 hook
- 5 秒讀懂測試：拿手機掃一眼，3 秒抓到「在講什麼 + 為何看」

## 失效開頭（對非追蹤者觸及衰減 40%+，換掉）

- 「這次真的殺瘋了」「各位！！！」「真的太神啦」「我又有新的 XX 了」
- 對**既有鐵粉**仍有效，想**擴散**一定換新開頭

## 高停留開頭（35-44 歲 AI 受眾實測前 5，取自 social-post）

1. 數字 + 動詞矛盾：「4,180 個追蹤者，昨天有 72,000 人看。」
2. 時間戳 + 事件：「凌晨 2 點按下發佈那一秒，我沒想到…」
3. 反問具體化：「如果你的 AI 會拒絕幫你寫 code，你會生氣還是感動？」
4. 公開認錯：「三個月前我說過的話，我想收回。」
5. 金額/時間成本：「這件事花了我 $47 + 6 小時，給你省掉兩個。」
