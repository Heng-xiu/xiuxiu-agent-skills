xiuxiu-agent-skills

給 AI 助手用的技能，
一件一件慢慢打磨。

技能（skill）是一份寫給 AI 助手的做事手冊：裝上之後，它處理特定任務時會照著一套驗證過的流程走，而不是每次即興發揮。

這裡收錄的每一個技能，都是從真實工作裡沉澱出來的。

網站與設計

網站拆解筆記site-teardown

給它一個網址，拿回一份看得懂的拆解筆記——長什麼樣、為什麼好看、背後怎麼做、接了哪些服務，落點是「下次跟客戶可以直接講的話」。

介紹頁 →

個人網站鍛造portfolio-forge

把你的個人資訊鍛造成一個美感極佳的個人網站：先畫出每一段的參考圖、逐張嚴格評審，再高保真做成真的網站、逐段驗收後上線。

介紹頁 →

工程品質

盤點未知數know-your-unknowns

動手改程式之前，先把「你不知道自己不知道」的事挖出來：盲點掃描、影響範圍訪談、決策日誌、合併前小測驗。

介紹頁 →

交辦前檢查delegation-review

把工作交給 AI 獨立完成之前，先檢查任務交代得夠不夠清楚：完成的標準寫了沒、該給的背景給了沒、卡住時它知不知道怎麼辦。

AI 伺服器體檢verify-ai-server-with-device-env

確認一台 AI 推論伺服器真的能在目標顯示卡環境上跑起來：驅動相容性、容器設定、模型載入、服務健康度，一關一關驗。

內容創作

內容產線content-pipeline

把一個粗略的想法，變成各平台可以直接發的社群貼文草稿，附發文建議。

怎麼安裝技能

把下面這段話貼給你的 AI 助手（例如 Claude Code），把「○○」換成上面任何一個技能的英文名字：

請幫我安裝 ○○ 這個技能（來源：GitHub 的 Heng-xiu/xiuxiu-agent-skills），裝好之後照它的流程帶我做一次。

習慣終端機的人：npx skills add Heng-xiu/xiuxiu-agent-skills --skill ○○，或在 Claude Code 裡 /plugin marketplace add Heng-xiu/xiuxiu-agent-skills
